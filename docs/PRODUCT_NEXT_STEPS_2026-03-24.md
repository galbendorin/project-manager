# Product Next Steps — 24 March 2026

## New request added

### Weekly timesheets tied to projects
- Add a post-login timesheet flow so a user can enter weekly time against existing projects.
- Treat this as a lightweight MVP first:
  - one weekly timesheet per user
  - project-linked entries
  - hours, notes, and week start date
  - simple weekly totals
- Keep the data model organisation-ready so it can evolve into a broader timesheet and reporting module later.

### Follow-on for organisation ownership
- Later iteration: support multiple owners or admins for the same organisation.
- That future model should sit above the current single-owner project model and should eventually govern:
  - organisation membership
  - shared project ownership
  - shared timesheet visibility
  - admin permissions and billing responsibility
